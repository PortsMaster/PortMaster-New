Thanks to `Jerome Labelle` and `Simon Labelle` for developing this game. Get it on [Steam](https://store.steampowered.com/app/718190/Koboo_The_Tree_Spirit/).

-----
The game suffers from a bug that can cause game being stuck on black screen after the end of the tutorial. Should that happen, you can still leave into the menu and begin a proper run from there.


## Controls

| Button | Action |
|--|--| 
|Dpad / Left Stick|Movement|
|A|Confirm/Jump|
|B|Dash/Cancel|
|X|Primary Weapon|
|Y|Secondary Weapon|
|R1|Interact|
|L1|Form Shift|
|L2/R2|Cast Abilities|
|Start|Options Menu|
|Select|Equipment Menu|
