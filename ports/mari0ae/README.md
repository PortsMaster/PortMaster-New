## Controls
| Button | Action |
|--|--| 
| D-Pad | Character movement/Menu navigation |
| A | Jump |
| B | Back\Pause |
| X | Sprint |
| Y | Use item |
| L1 | Fire portal 1 |
| R2 | Fire portal 2 |
| L2 | Reset portals |
| R2 | Use item |
| Select | Back\Pause |
| Start | Confirm |
| Left Analog | Point portal gun |
