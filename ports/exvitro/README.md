## Controls

| Button | Action |
|--|--| 
|Dpad / Left Stick|Movement / Selection|
|A|Jump|
|B|Run|
|X|Fire beam|
|Y|XMP|
|L1/R1|Change Weapons|
|L2/R2|Aim at angles|
|L3|Auto-crouch|
|R3|Auto-sprint|
|L2 + Start|Save game|
