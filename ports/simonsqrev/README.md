## Controls

| Button | Action |
|--|--| 
|start|start|
|select|exit/quit|
|d-pad|movement|
|left-analog|movement|
|B|whip|
|Y|whip/sub weapon|
|A|jump|
|X|jump|
|R2|inventory|
|L2|map|
|L1|choose sub item|
|R1|swap|
