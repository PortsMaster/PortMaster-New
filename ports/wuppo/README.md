| Button | Action |
|--|--|

|L2|Change Target L|
|R1|Use Item|
|R2|Change Target R|
|Up|Progress Window|
|Down|Hold Money|
|Left|Filmstrips|
|Right|Items|
|Start|Pause / Minimap|
|Select|Options|
