## Controls

| Button | Action |
|--|--| 
|DPAD / Left Stick|Move|
|A/B/X/Y|Shoot|
|L1/L2|Focus (slow-mo) (EX only)|
|R1/R2|Hyper Attack|
|R1 (at the start of the level)|Change ship (EX only)|
|Start|Pause|
