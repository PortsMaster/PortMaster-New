## Controls

| Button | Action |
|--|--| 
|D-pad / L-stick|Movement|
|R-stick|Aim / Mouse|
|L1 / L2|Melee|
|R1 / R2|Shoot / Select|
|A|Interact|
|B|Jump|
|X / Y|Toggle Weapons|
|Start / Select|Pause / Menu|
