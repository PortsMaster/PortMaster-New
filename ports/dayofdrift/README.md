## Controls

| Button | Action |
|--|--| 
|Dpad / Left stick|Menu controls / Steering|
|A|Confirm / Handbrake|
|B|Cancel / NOS|
|R1/R2|Accelerate|
|L1/L2|Reverse|
|Start|Menu|
|Select|Restart|
