## Controls

| Button | Action |
|--|--| 
|DPAD / Left Stick|Movement|
|Right Stick|Shoot|
|A|Jump / Confirm|
|B|Cancel|
|X|Melee / Drop item|
|R1|Action|
|L1|Magic Item|
|L2|Quickslot|
|Select|Inventory|
|Start|Pause menu|
