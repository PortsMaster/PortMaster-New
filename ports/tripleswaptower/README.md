## Controls

| Button | Action |
|--|--| 
|D-pad / L-stick|Movement|
|North|Swap Slot 1|
|South|Select / Dash|
|East|Cancel / Swap Slot 2|
|West|Attack|
|Shoulder L / R|Swap Heroes|
|Trigger R|Swap Slot 3|
|Start|Pause / Menu|
