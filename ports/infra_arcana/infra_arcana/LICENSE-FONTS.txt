GENERAL INFORMATION

This file lists all font images which are stored in the Infra Arcana
source code repository (https://gitlab.com/martin-tornqvist/ia), and
distributed with executable builds of the project. For each font image
file, an attribution is included below (source, original title, license,
and whether any significant changes have been made).

All fonts included with Infra Arcana have been converted from TTF
(TrueType Front) files, into PNG (Portable Network Graphics) images.
