## Controls

| Button | Action |
|--|--| 
|A|Interact / Jump|
|B|Cancel / Back|
|Y|Bicycle|
|DPAD|Movement|
|L-STICK|Alt. Movement|
|R-STICK|Battle inputs|
|Back buttons|Alt. battle inputs|
|Start|Menu|
|Select|Interact|
|R3|Q battle input|
