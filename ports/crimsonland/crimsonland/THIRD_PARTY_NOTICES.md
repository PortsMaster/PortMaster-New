# Crimsonland PortMaster Notices

Crimsonland is the original game by 10tons Ltd. This PortMaster package is an
independent community build of the native Zig Crimsonland reimplementation from
the Crimson rewrite project. It is not an official 10tons release.

This package is intended to ship without original Crimsonland `.paq` archives.
Users should provide their own `crimson.paq` and `sfx.paq`; `music.paq` is
optional for this port.

The upstream Crimson rewrite README states that original Crimsonland Classic
assets are distributed for that project with permission from the original
developer. Do not assume that permission covers third-party PortMaster release
zips unless you have separately verified redistribution rights.

PortMaster packaging, launcher, controller mapping, and handheld UI adjustments
in this branch are downstream changes for small-screen handheld devices.
