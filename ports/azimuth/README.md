## Controls

| Button | Action |
|--|--| 
|DPAD| Move |
|A | Shoot|
|B | Ordnance| 
|Y | Utility | 
|Back | Pause / Map |
|L1 | Mouse Left |

You can find the manual for this game here:
https://mdsteele.games/azimuth/manual.html#controls

## Building

```
git clone https://github.com/mdsteele/azimuth.git
cd azimuth
BUILDTYPE=release make 
```
