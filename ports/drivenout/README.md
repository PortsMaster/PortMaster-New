## Controls

| Button | Action |
|--|--| 
|DPAD / Left Stick|Walk|
|A|Accept|
|B/X/Y|Attacks|
|R1|Drop Device|
|R2 + B/X/Y|Blocking|
|L1|Turn|
|L2|Run|
|Start|Pause menu|
