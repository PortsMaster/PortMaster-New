## Controls

| Button  | Action                        |
| ------- | ----------------------------- |
| L-STICK | Mouse cursor                  |
| X       | Mouse click                   |
| D-PAD   | (Some of the) menu navigation |
| L1/L2   | Left flipper                  |
| R1/R2   | Right flipper                 |
| Y       | Table bump                    |
| SELECT  | Options menu                  |
