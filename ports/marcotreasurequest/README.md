## Controls

| Button | Action |
|--|--| 
|D-Pad / Joystick|Move|
|A|Interact / Confirm|
|B|Menu / Back|
|Y|Run|
|Right Joystick Up|W|
|Right Joystick Down|S|
|Right Joystick Left|A|
|Right Joystick Right|D|
