## Controls

| Button | Action |
|--|--| 
|D-pad|Movement|
|A|Accept|
|B|Roll / Cancel|
|X|Reload|
|Y|Interact|
|L1|Swap Item|
|L2|Item|
|R1|Swap Weapon|
|R2|Attack|
|Start|Inventory|
|Select|Pause|
