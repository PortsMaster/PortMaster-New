## Installation
Download the Windows executable from https://dragonrod342.itch.io/digital-tamers-reborn and put in the `ports/digitamersreborn/assets` folder.

## Controls
| Button | Action |
|--|--|
|ANALOG|Move Cursor|
|A|Left Click|
|B|Right Click|
