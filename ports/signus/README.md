## Controls

| Button | Action |
|--|--| 
|DPAD| Move|
|Analogue Stick| Move Mouse|
|Select| Menu|
|Start| Space (Start Game)|
|A| Left Mouse|
|B| Right Mouse|
|X| Slowdown Mouse|
|Y| Space|
|L1| Game Menu|
|L2| Save Menu|
|R1| Load Menu|
```

See [BUILDING.md](https://github.com/PortsMaster/PortMaster-New/blob/main/ports/signus/signus/BUILDING.md) for building instructions.
