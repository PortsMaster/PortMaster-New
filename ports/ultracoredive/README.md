## Controls

| Button | Action |
|--|--| 
|Dpad|Menu controls|
|A/B|Confirm/Cancel|
|Left stick|Movement|
|Right stick|Shoot|
|L2|Dodge|
|R2|Special|
|Start|Pause menu|
