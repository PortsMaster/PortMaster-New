
## Controls

| Button | Action |
|--|--| 
|Jump|Y|
|Umbrella|R Trigger|
|Walk|R Bumper|
|Info|X|
|Hologram|A or press Down twice|
|Map|L Trigger|
|Pause|Start|
|Die|Select|
|UI Accept|Y, B, A|
|UI Back|X|
|UI Page Up|L Bumper|
|UI Page Down|R Bumper|
|Map +/-|Umbrella / Walk|
|Map Marker|Jump|
|Console|Info + Pause|
|Adjust Height|Info + Umbrella + Left/Right|
|Emergency Save|Info + Umbrella + Down|
|Emergency Exit|Start + Select|

If you are experiencing lags at save points or screen transitions, you can turn off detailed maps in the settings.
