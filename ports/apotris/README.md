## Controls

| Button | Action |
|--|--|
| Dpad | Move Pieces |
| DPAD UP | Drop Piece |
| A/B | Rotate Piece |

## Compiling

```bash
git clone -b rgb30 https://gitea.com/akouzoukos/apotris.git
git submodule update --init --remote --force
make rgb30
```
