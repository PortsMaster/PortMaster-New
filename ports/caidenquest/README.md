## Controls

| Button | Action |
|--|--| 
|Dpad / Left stick|Movement|
|A|Jump / Confirm|
|B|Cast spell|
|X|Boost|
|L1 / R1|Spell select / Menu scrolling|
|Select / Start|Menu|
