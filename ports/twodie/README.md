## Controls

| Button             | Action |
|--------------------|--| 
| DPAD/LSTICK/RSTICK |Move, menu navigation|
| X/A/L2/R2          |Confirm, select menu item|
| Select/Y/B/L1/R1   |Cancel, open menu|
