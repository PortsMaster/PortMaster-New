## Notes

Rising Dusk (Steam App 848930) is a paid game by Studio Stobie. You must own a copy on Steam and copy the game files yourself — see the installation instructions.

This port runs the original Windows executable through Wine + Box64 and requires a CFW that ships these tools (ROCKNIX or JELOS/KNULLI builds that include Wine and Gamescope). It will not work on muOS, ArkOS, or other CFWs that do not provide Wine.

A preparation script (`prepare_port.py`) is included to automatically find and copy the required files from your Steam installation on Windows, macOS, or Linux.

Port by [Clara Ogalla Moreno (Crazyx98)](https://github.com/clarax98).
