## Controls

Movement is on dpad by default, but can be set to stick in the settings.

| Button | Action |
|--|--| 
|DPAD |Movement|
|A|Jump/Confirm|
|B|Action/Cancel|
|X|Ability 1|
|Y|Ability 2|
|L1|Slide|
|R1|M. Scratch|
|L2/R2|Ability cycle|
|Start|Pause Menu|
|Select|Screenshot|
