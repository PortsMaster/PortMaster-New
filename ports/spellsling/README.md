## Controls

| Button | Action |
|--|--| 
|DPAD/LSTICK/RSTICK|Move, menu navigation|
|A|Confirm, use|
|B|Spell info|
|L1/L2/Y|Prev spell|
|R1/R2/X|Next spell|
