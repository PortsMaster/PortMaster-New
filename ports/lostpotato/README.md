## Controls

| Button | Action |
|--|--| 
|D-pad / L-stick|Movement|
|R-stick / ABXY|Aim|
|L1 / L2 / R1 / R2|Hit|
|Start|Pause / Menu|
|South|Select|
|East|Back|

If ABXY directions are flipped, delete `a b` and `x y` in  lostpotato/SDL_swap_gpbuttons.txt
