## Controls

| Button | Action |
|--|--| 
|Dpad|Meta Tree scrolling|
|Left Stick|Mouse|
|A|Left mouse button|
|B|Right mouse button|
|X|Mouse slow|
|R1|Toggle dpad as mouse|
|Start|Pause menu|
