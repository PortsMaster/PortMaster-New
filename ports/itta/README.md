## Controls

| Button | Action |
|--|--| 
|L-stick |Movement|
|R-stick|Aim|
|L1/R1|Switch Weapon|
|L2|Dodge|
|R2|Shoot|
|A|Select / Interact|
|B|Map|
|Start|Pause|
