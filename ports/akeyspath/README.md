## Controls

| Button  | Action                                                           |
| ------- | ---------------------------------------------------------------- |
| D-PAD   | Player movement                                                  |
| L-STICK | Mouse cursor (OR touch-screen input if your device support this) |
| A       | Mouse hold/release                                               |
| B       | Jump                                                             |
| Y       | Retrieve all keys                                                |
| SELECT  | Menu/pause                                                       |
