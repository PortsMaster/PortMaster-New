## Controls

| Button | Action |
|--|--| 
|DPAD| Steer Board|
|A| Thrust|
|B| Reverse Thrust|
|Y| Jump|
|X| Turn Around|
|L1/R1 | Zoom in/out|
|L2| Pause|

git clone https://github.com/Cebion/bloboats_pm
