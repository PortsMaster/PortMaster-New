## Notes

This is a simple mod manager for Stardew Valley on these retro handhelds. This is my first tool I have made, but I haven't ran into any problems. This tool also allows for multiple mods to be enabled, and keeps track of which ones are enabled, and which files are changed.

## Controls

| Button | Action |
|--|--| 
|D-Pad|Navigation|
|A|Select|


## Compile

```shell
Make sure you have the Stardew Valley port installed, and haven't
 replaced any assets, as this tool will make a backup of the vanilla game, and make a mods
 folder. After the tool finishes making the backup, you can start adding 
 mods to the mods folder. I have only tested it with sprite 
 swaps (XNB files), and I don't think it will work with more 
 complicated mods, so no custom menus with more options. 
```
