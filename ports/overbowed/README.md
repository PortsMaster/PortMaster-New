## Controls

| Button | Action          |
| ------ | --------------- |
| D-PAD  | Move            |
| A      | Jump/interact   |
| X      | Shoot           |
| START  | Pause/item menu |
| SELECT | Map             |
| L1     | Aim/focus       |
