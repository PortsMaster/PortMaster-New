<title>Languages</title>

<h1><strong>Languages</strong></h1>

<p>
The <strong>Languages</strong> window changes the language used in the game.
</p>
<p>
It opens from the <a href="options.txt">Game Options</a> window and with the <a href="new_world.txt">Create a New Game</a> windows.
</p>
<p>
The default language for Simutrans is English. To play the game in
a different language, select the button next to the language you would like to play in.</p>
<p>
Language can be changed any time during game play. To change the language, select Game Options -> Language.</p>
<p>Language change happens immediately when creating a new game and on closing the <strong>Languages</strong> window when changing mid game.
</p>
<p>
If a translation is not available in the selected language, it may appear in English.
</p>

<p>
To help in translating Simutrans in your language,  <br>
visit https://translator.simutrans.com/
</p>
