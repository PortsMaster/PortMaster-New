<title>Einstellungswerkzeuge</title>

<h1><strong>Einstellungen, Laden, Speichern</strong></h1>

<p>
Hier findest Du erst all Werkzeuge zum <a href="save.txt"><em>Speichern</em></a> und zum Start eines neuen Spiels, entweder vom <a href="new_world.txt"><em>Computer generiert</em></a>, ein zuvor <a href="load.txt"><em>gespeichertes Spiel laden</em></a>, eines von einem Server oder eine geskriptete <a href="scenario.txt"><em>Mission</em></a>.<br>
Dahinter folgen Enstellungsfenster für die <a href="display.txt"><em>Anzeige</em></a>, den <a href="sound.txt"><em>Sound</em></a>, die <a href="language.txt"><em>Anyeigesprache</em></a>, die <a href="color.txt"><em>Spielerfarbe</em></a> in der viele Fahrzeuge und Halte erscheinen, der alte <a href="options.txt"><em>Einstellungfenster</em></a>, die erweiterten <a href="settings.txt"><em>Einstellungen</em></a> für Profis.<br>
Zum Schluss kommen diverse Schnellzugriffe auf Anzeigeinstellungen, wie Zoom, Gitternetz, Untergrundmodi, Sie Anzeige von Spieler oder Reservierungen, oder dem Ausblenden verschiedener Objekte.</p>
