<title>Setting Toolbar</title>

<h1><strong>Save, Load, Settings</strong></h1>

<p>
At the start are the tools for <a href="save.txt"><em>saving</em></a> and to begin a new games, either 
<a href="new_world.txt"><em>computer generated</em></a>, <a href="load.txt"><em>loading a saved game</em></a>, join a server game, or start a scripted <a href="scenario.txt"><em>mission</em></a>.<br>
Next follow the various setting windows for the <a href="display.txt"><em>display</em></a>, the <a href="sound.txt"><em>sound</em></a>, the <a href="language.txt"><em>display language</em></a>, the <a href="color.txt"><em>player colour</em></a> in which many vehicles and stop will appear, the old <a href="options.txt"><em>setting window</em></a>, an the <a href="settings.txt"><em>extended settings</em></a> for working deep in the simutrans inyards.<br>
The rest of the toolbar are quick access on various visual settings, like zoom, underground modes, information overlays or hinding objects.</p>
