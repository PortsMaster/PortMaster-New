<h1>Description pak64.german</h1><p>The pak64.german is a graphics set that only uses one height.</p><p>In the pak64.german there is no bonus for speed. But this is still important because stations must not be overcrowded. If stations are overcrowded, the transport volume collapses.<br>If minimum load is used, make sure that stations are not overcrowded.</p><p>Magnet railway and the S-Bahn/U-Bahn should only be used when passenger volumes are very high.</p>
