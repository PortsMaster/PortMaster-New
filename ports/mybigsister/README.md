## Controls

| Button | Action |
|--|--| 
|Dpad / Left stick|Movement|
|A|Action|
|B|Cancel (Remaster only)|
|X|Inventory|
|Y|Run (Remaster only)|
|Start / Select|Menu / Skip cutscene|
