## Controls

| Button  | Action      |
| ------- | ----------- |
| D-PAD   | Movement    |
| L-STICK | Mouse move  |
| L3      | Mouse slow  |
| X/A     | Snap/place  |
| Y       | Cancel      |
| B       | Undo        |
| R1      | Reset       |
| L1      | Skip level  |
