## Controls

| Button | Action |
|--|--| 
|D-pad/L-stick|Movement |
|A|Menu selection/New game|
|B|Menu selection/New game|
|Start|Menu selection/New game|
|Select|Main Menu/Quit|
|R2|Main Menu/Quit|
|L2|Fullscreen button if in DE (SteamDeck, etc.)|
