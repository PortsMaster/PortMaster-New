## Notes

Thanks to [Koliao](https://koliao.itch.io) for creating this little gem, and for granting PortMaster permission to distribute the game files.

**RK3326 users please note** that this game runs best on more powerful devices.

**ROCKNIX users please note** that this port does not support the Panfrost graphics driver. Switch to libMali to play.


## Controls

| Button | Action          |
| ------ | --------------- |
| D-PAD  | Move            |
| A/B    | Jump            |
| X      | Croak           |
| Y      | Hold to restart |
