## Controls

| Button | Action                                   |
| ------ | ---------------------------------------- |
| D-PAD  | Directional movement                     |
| A      | Investigate / advance / confirm          |
| B      | Raise menu / cancel choice / flee battle |
| X      | Sprint on the *field* when held down     |
| Y      | Pause during battle                      |

## Compile OHRRPGCE 

```shell
Install FreeBASIC
git clone https://github.com/ohrrpgce/ohrrpgce.git
cd ohrrpgce
scons gfx=sdl2
```
