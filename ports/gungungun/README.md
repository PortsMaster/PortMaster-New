## Controls

| Button | Action |
|--|--| 
|D-pad / L-Stick|Move|
|R-Stick|Aim|
|A|Jump|
|X|Reload|
|L2| Draw Gun (Need to draw before aiming)|
|R2|Fire|
|Start|Pause / Menu|
