## Installation
Buy the game on Steam or Itch.io and copy all data to `haque/assets`.

## Default Gameplay Controls
| Button            | Action                                |
|--                 |--                                     |
| START             | Menus                                 |
| SELECT            | Wait                                  |
| D-PAD / JOYSTICK  | Move                                  |
| A                 | Confirm                               |
| B                 | Deny                                  |
| X                 | Backpack                              |
| Y                 | Ally AI                               |
| L1                | Shift                                 |
| L2                | Quick Skill                           |
| R1                | Map                                   |
| R2                | Quick Item                            |
