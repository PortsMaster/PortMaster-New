## Controls

|Button| Action |
|--|--|
| d-pad/l-stick| movement |
| r-stick| aim |
| a| melee |
| b/l1| dodge |
| x/r1| shoot |
| y| swap player |
| start| pause/menu |
| l2/r2| jump (platforming mode) |
