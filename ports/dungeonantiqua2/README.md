## Notes

Thanks [Shiromofu Factory](https://dungeon-antiqua.com/) for making this game.

## Language
By default the game will be in Japanese, but you can change this in the config menu when starting the game.

## Controls

| Button | Action |
|--|--| 
|DPAD|Move, cursor selection|
|A|OK|
|B|Cancel|
|X|Open Menu|
|Y|Open Menu|
|Start|Open Game Menu|
