## Controls

| Button | Action |
|--|--| 
|D-pad / L-stick|Movement|
|R-stick|Mouse / Aim|
|R1 / R2|Shoot|
|L1 / A / B|Shank|
|X / Y|Change Weapon|
|Start|Enter|
|Select|Escape|
