## Controls

| Button  | Action                   |
| ------- | ------------------------ |
| D-PAD   | Movement                 |
| L-STICK | Mouse move               |
| X       | Mouse click              |
| A       | Advance / Remaining Fuel |
| B       | Undo                     |
| R1      | Reset                    |
| L1      | Skip level               |
