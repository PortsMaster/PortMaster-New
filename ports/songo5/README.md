## Controls
| Button | Action |
|--|--| 
|DPAD / Left stick|Nav through UI / Skip song / Previous Song|
|L1 / R1 |Skip and previous song, usable even when navigating other menus|
|A|Select UI item / pause song / resume song|
|B|Nav back in UI|
|X|Shuffle play if viewing a song list|
|Y|Open Quick Menu|
|Start|Dismiss currently playing song if 'Song follow' is enabled|
|Select|Return to currently playing song if 'Song follow' is enabled|
