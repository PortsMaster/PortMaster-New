## Controls

| Button | Action |
|--|--| 
|Left Analog|Red Guy move|
|Right Analog|Blue Guy move|
|D-Pad|Green Guy move|
|X|Yellow Guy up|
|B|Yellow Guy down|
|Y|Yellow Guy left|
|A|Yellow Guy right|
|Start|Pause|
