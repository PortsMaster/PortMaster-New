## Controls

| Button | Action |
|--|--| 
|DPAD / Left stick|Mouse|
|A|Left mouse button|
|B|Right mouse button|
|X|Slow mouse|
|DPAD down + Start|Interactive text input (for naming saves)|
|Start|Menu|
