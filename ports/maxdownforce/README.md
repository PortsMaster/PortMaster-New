## Controls

| Button | Action |
|--|--| 
|DPAD/LSTICK/RSTICK|Stear|
|R1/R2/A|Accelerate|
|L1/L2/B|Break|
|X|Change controls schema|
|Y|Change racing track|
|Select+Y|Mute music|
|Select|Exit|
|Start|Start race|
