## Controls

| Button | Action |
|--|--| 
|D-pad / Left stick|Move|
|A|Start game / Jump|
|B|Attack|
|Y|Dash|
|X|Special (needs Torque)|
|R1/L1|Change weapon|
|Start|Pause|
|Start + Select|Exit game|
