## Controls

**Note for controls:** some hotkeys are available only in the full version.


| Button | Action |
|--|--| 
|Dpad|Map scrolling|
|Left stick|Mouse|
|A|Left mouse|
|B|Right mouse|
|X|Mouse slow|
|Y|Build menu|
|R1|Toggle dpad for mouse controls|
|R2|Secret tile counter|
|L1 / L2|Zoom|
|Start|Menu|
