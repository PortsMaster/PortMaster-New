## Controls

| Button | Action |
|--|--| 
|A| Left Mouse|
|B| Right Mouse|
|X| Slow Mouse Movement|
|Y| Space |
 

## Compile

```shell
git clone https://github.com/the3dfxdude/7kaa.git
./autogen.sh
./configure
make
```
