## Controls

| Button | Action |
|--|--| 
|DPAD / Left stick|Movement|
|Right stick|Teleport|
|DPAD / Left stick + R2|Teleport (alternative bind)|
|A|Main weapon / Confirm|
|B|Sub weapon / Back|
|L1|Half speed|
|L2|Focus movement|
|Start|Menu|
