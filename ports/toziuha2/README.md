## Notes

Thanks to Danny Garay for developing the game and also doing the initial porting work. Get his games on the official stores, links [here](https://dannygaray60.github.io) 

---

Note: the demo version suffers from graphical glitches on libmali driver under Rocknix for RK3326/3566 units, use Panfrost driver there. Full version is not affected. 

## Controls

| Button | Action |
|--|--| 
|DPAD / Left stick|Movement|
|A|Confirm / Jump|
|B/X|Cancel / Attack|
|L1|Dodge|
|R1|Elemental circuit|
|Start|Pause/Equipment|
|Select|Map|
