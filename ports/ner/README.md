## Controls

| Button | Action |
|--|--| 
|DPAD/LSTICK|Move, menu navigation|
|RSTICK|Look|
|A|Confirm|
|B/SELECT/START|Exit, cancel|
|Y|Use compas|
|X|Inspect compas|
|L1/L2|Lean left|
|R1/R2|Lean right|
