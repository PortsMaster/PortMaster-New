## Controls

| Button  | Action                         |
| ------- | ------------------------------ |
| D-PAD   | Movement                       |
| L-STICK | Mouse move (for level select)  |
| X       | Mouse click (for level select) |
| A       | Continue                       |
| B       | Undo                           |
| R1      | Reset                          |
| L1      | Mute                           |
| SELECT  | Exit to level select menu      |
