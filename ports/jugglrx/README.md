## Controls

| Button | Action |
|--|--| 
|DPAD/LSTICK/RSTICK|Move|
|A/X|Spawn ball|
|B/Y|Use skill|
|L1/L2/R1/R2|Change paddle 1-4|
|Select|Pause|
|Start|Restart|
|Select+Y|Toggle music|
|Select+X|Toggle sfx|
