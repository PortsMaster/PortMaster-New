INGAME KEYS

Arrows ---- Move around
Z --------- Ground fire
X --------- Air fire
Esc ------- Pause menu

Control + Shift + R while in pause menu 
will reset the configuration file.

Joypads are also supported!
Check the pause menu to disable them.

-----------------------------------------

Configuration and hiscore files are
stored in your local App data folder,
not in the same folder of the program.
