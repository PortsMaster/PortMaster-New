## Controls

| Button | Action |
|--|--|
| Dpad | Move |
| Left Analog | Move |
| Right Analog | Move |
| A | Start/wait |
| B | Inventory |
| X | Examine |
| Y | Interact (Equip/Activate) |
| L1 | Settings/Help |
| L2 | Message |
| R1 | Quit |
| R2 | Use as Y to confirm |
## Compiling

go build --tags sdl
