# Thanks
Thanks to [Jarmo Hekkanen & Lauri Hakkarainen](https://github.com/Cebion/oilwar) for creating this game and making it available for free.

## Controls

| Button | Action |
|--|--| 
|DPAD| Move |
|A | Shoot Gun|
|B | Drop Bomb| 

## Building

```
git clone https://github.com/Cebion/oilwar.git
cd oilwar
./configure
make
```
