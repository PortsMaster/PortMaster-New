## Notes

Download the game directly from the [itch.io](https://misaeuu.itch.io/barrelroll) website. Extract the files and place them in the ports/barrelroll/assets directory.

Special thanks for [misaeuu](https://misaeuu.itch.io/barrelroll) for making this wonderful game!


## Controls

| Button | Action |
|--|--| 
|D-PAD / LEFT JOYSTICK|Move|
|B|Undo|
|Y|Reset|
|SELECT / START|Pause|
