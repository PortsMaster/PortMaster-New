## Controls

| Button        | Action                        |
| ------------- | ----------------------------- |
| L-STICK/D-PAD | Mouse                         |
| A             | Click (select/perform action) |
| B             | Clear current action          |
| Y             | Dismiss dialogue box          |
| SELECT        | Pause menu                    |

## Action Shortcuts

| Button | Action  |
| ------ | ------- |
| L1     | Examine |
| L2     | Take    |
| X      | Move    |
| R1     | Use     |
| R2     | Open    |
