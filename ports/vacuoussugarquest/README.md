## Notes

Thanks to [Walfie](https://walfie.itch.io) for creating this little gem, and for granting PortMaster permission to distribute the game files.

Estimated length: probably between 40 to 90 minutes, depending on your platforming/puzzle-solving abilities and whether you want to 100% it.


## Controls

You can remap controls in the Settings menu.

| Button | Action   |
| ------ | -------- |
| D-PAD  | Movement |
| A      | Jump     |
| BACK   | Reset    |
