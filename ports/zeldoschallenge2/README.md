## Installation
Download the game from https://solarus-games.org/games/defi-de-zeldo-ch-2/ and put in the `ports/zeldoschallenge2` folder.

## Default Gameplay Controls
| Button | Action |
|--|--|
|D-PAD|Move|
|A|Interact|
|B|Attack|
|X|Item 1|
|Y|Item 2|
|START|Menus|
