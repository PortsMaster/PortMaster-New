## Controls

| Button | Action |
|--|--|
| Dpad | Move |
| Left Analog | Move|
| Right Analog | Move |
| A | Dig Left |
| B | Dig Right |
| X | Space/Start |
| Y | Pick Level |
| L1 | Go to Previous Level  |
| L2 | Go to Next Level |
| R1 | Goto |
| R2 | FPS Counter |

## Compiling

make

[Source code](https://github.com/monkeyx-net/SDL-Scavenger) changed to allow to run more efficiently and added fps counter.
