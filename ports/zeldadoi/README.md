## Controls

| Button | Action |
|--|--| 
|A|Run (hold)|
|B|Sword|
|Y|Item|
|X|Map|
|Start|Inventory / Map|
|Select|Game Menu|
|D-Pad|Movement|
|Left Analog|Movement|
