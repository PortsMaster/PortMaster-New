## Controls

| Button        | Action             |
| ------------- | -------------------|
| D-PAD         | Movement           |
| L-STICK (L/R) | Mouse/Movement     |
| B             | Jump               |
| SELECT        | Pause/Menu         |
| START         | Enter/Select Option|
