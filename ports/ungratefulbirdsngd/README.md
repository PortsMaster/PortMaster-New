## Controls

| Button | Action              |
| ------ | ------------------- |
| D-PAD  | Move                |
| A      | Jump                |
| X      | Pick up / throw     |
| Y      | Indicators          |
| B      | Rescue (co-op mode) |
| START  | Menu                |
