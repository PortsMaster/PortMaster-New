## Controls

| Button | Action |
|--|--| 
|Dpad|Menu / Inventory selection|
|Left stick|Movement|
|A|Use|
|B|Running|
|X|Dodge|
|Y|Healing|
|L1|Gun|
|L2|Guard|
|R1|Light attack|
|R2|Heavy attack|
|L3|Lamp|
|Start|Pause menu|
|Select|Inventory|
