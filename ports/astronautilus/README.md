## Controls

One can remap controls in the Options menu.

| Button  | Action             |
| ------- | ------------------ |
| L-STICK | Move               |
| R-STICK | Aim                |
| A       | Interact           |
| B       | Skip               |
| X       | Bomb up            |
| Y       | Bomb down          |
| R2      | Fire / Mouse-click |
| L2      | Dash               |
| L1      | Bomb               |
