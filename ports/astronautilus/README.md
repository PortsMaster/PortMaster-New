## Controls

| Button  | Action             |
| ------- | ------------------ |
| L-STICK | Move               |
| R-STICK | Aim                |
| A/B     | Interact           |
| R1      | Mouse-click / Fire |
| R2      | Bomb               |
| R3/L1   | Dash               |
| START   | Quick-skip         |
