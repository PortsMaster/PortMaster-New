## Controls

**For the demo version:** there's no bomb up/down; avoid using L-Stick to navigate menus (use 'mouse' instead).

One can remap controls in the options menu.

| Button  | Action             |
| ------- | ------------------ |
| L-STICK | Move               |
| R-STICK | Aim                |
| A       | Interact           |
| B       | Skip               |
| X       | Bomb up            |
| Y       | Bomb down          |
| R2      | Fire / Mouse-click |
| L2      | Dash               |
| L1      | Bomb               |
