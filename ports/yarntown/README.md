## Installation
Download game from https://solarus-games.org/games/yarntown/ and put the .solarus file in `ports/yarntown`.

## Default Gameplay Controls
| Button | Action |
|--|--|
|D-PAD|Move|
|A|Action / Dodge|
|B|Swing weapon|
|B (Hold)|Charge weapon|
|X|Shoot pistol|
|Y|Heal|
|START|Pause & Save|
