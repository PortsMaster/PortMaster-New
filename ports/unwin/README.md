## Controls

| Button | Action |
|--|--| 
|DPAD / Left stick|Movement / Selection|
|A|Accept / Unwin|
|B|Cancel / Undo|
|X|Really Undo|
|Y|Really Redo|
|Start|Menu|
|Select|Restart|
