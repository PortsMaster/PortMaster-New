## Controls

| Button | Action |
|--|--| 
|DPAD| Player 1 Move |
|L1| Player 1 Shoot |
|L2| Player 1 Eject |
|A/B/X/Y | Player 2 Move|
|R1| Player 2 Shoot |
|R2| Player 2 Eject |

## Building

```
git clone --recurse-submodules https://github.com/regular-dev/biplanes-revival
cd biplanes-revival
mkdir build && cd build
cmake .. 
make
 ```
