## Controls

| Button | Action |
|--|--| 
|D-pad|movement|
|L-analog stick|movement|
|A-button|jump/start|
|X-button|fire/chew|
|Y-button|fire/chew|
|B-button|fire/chew|
|L2|fire/chew
|R2|fire/chew
|Start-button|Start|
