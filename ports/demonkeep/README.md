## Controls

| Button | Action |
|--|--| 
|DPAD/LSTICK/LSTICK|Move|
|A|Use item|
|B|Cancel|
|Y/L1/L2|Prev item|
|X/R1/R2|Next item|

|Select |Toggle help|
|Start|Restart (after death)|
