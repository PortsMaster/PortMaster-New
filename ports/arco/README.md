## Controls

| Button | Action |
|--|--| 
|D-pad|Menu Movement/ Skills Select|
|L-Stick|Character Movement|
|A|Select|
|B|Back / Cancel|
|X|Management Menu / Purchase|
|Y|Switch Characters|
|L / R|Menu / Skills Select|
|R2|Combat Preview|
|L2|D-pad/L-stick swap for D-pad only devices|
